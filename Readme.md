# ProductsApp

## Запуск

```bash
git clone https://github.com/aszxqaz/products-app-extjs.git
cd products-app-extjs
sencha app watch
```

Если `sencha` автоматически не подгружает фреймворк:

```bash
sencha framework add ext@6.0.0
```

### Данные для входа:

Логин _admin_
Пароль _padmin_
